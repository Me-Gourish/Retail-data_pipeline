🚀 Built an End-to-End Data Pipeline!

I designed and implemented a complete data pipeline using Snowflake, dbt, Apache Airflow, and Power BI.

🔹 Data Warehouse: Snowflake
🔹 Transformation: dbt (staging, intermediate, mart layers)
🔹 Data Quality: dbt tests
🔹 Historical Tracking: dbt snapshots
🔹 Orchestration: Airflow DAG
🔹 Visualization: Power BI dashboard

This project helped me understand real-world data engineering workflows including data modeling, pipeline automation, and analytics.
